Under the Hood

Technical specifications for power users.

0-bit
AES-CBC Security

Military-Grade Security

  • Algorithm: AES-256 CBC
  • Key Derivation: SHA-256 hash of your 4-digit PIN
  • Initialization Vector: 16-byte random IV per packet
  • Padding: PKCS7 standard padding

Lossless Capture

  • Sample Rate: 0 Hz
  • Bit Depth: 0-bit
  • Channels: Mono (Microphone Standard)
  • Codec: Raw PCM (uncompressed, no artifacts)

Latency: ~20ms

Real-Time Streaming

  • Protocol: UDP (User Datagram Protocol)
  • Port: 55555 (default, configurable)
  • Packet Size: Optimized for MTU limits
  • Error Handling: Automatic drop of out-of-order packets

Broad Compatibility

  • Windows: 10 / 11 (64-bit)
  • Android: 10.0 or higher
  • Network: Same local Wi-Fi or USB tethering
  • Audio Driver: VB-Audio Virtual Cable (included)

Fully Open Source

Built with Python and Flutter. No trackers, no ads, no telemetry.
